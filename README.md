# Dashboards PowerBI

## Objetivos
Elaboração de dashboards com o intuito de aprender e desenvolver a capacidade de utilização da ferramenta Power BI e de habilidades analíticas. Foram construídos dashboards de diferentes áreas, que são apresentadas abaixo:
- [Dashboard de dados de Marketing](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/ProjetoMarketingDSA.pbix)
- [Dashboard de dados Comerciais](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/ProjetoComercialDSA.pbix)
- [Dashboard de dados de Recursos Humanos](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/ProjetoRHDSA.pbix)
- [Dashboard de dados de Logística](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/ProjetoLog%C3%ADsticaDSA.pbix)
- [Dashboard de dados Financeiros](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/ProjetoFinanceiroDSA.pbix)
- [Dashboard de dados Contábeis](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/Balan%C3%A7oPatrimonialDSA.pbix)

## Dashboard de dados Contábeis
#### Dashboard
![image](https://github.com/mathwatanabe/Dashboards-PowerBI/assets/170628618/3defc86c-b98b-4e9c-8261-e416665e4871)

Acesse o dashboard completo [neste link](https://github.com/mathwatanabe/Dashboards-PowerBI/blob/main/Balan%C3%A7oPatrimonialDSA.pbix)

### Estrutura da base de dados

Arquivo possuía duas tabelas. Uma tabela de Dados Contábeis com o valor de fechamento de cada ano. Uma tabela com a identificação de cada conta e seus respectivos tipos. Na tabela de Dados Contábeis estavam as seguintes colunas:
- ID da Conta (Código de Identificação da Conta)
- Valor na Conta 2019
- Valor na Conta 2020
- Valor na Conta 2021
- Valor na Conta 2022
- Valor na Conta 2023

Na tabela Planos de Contas estavam as seguintes colunas:
- ID da Conta (Código de Identificação da Conta)
- Tipo de Relatório
- Conta Nível 1 (Ativo ou Passivo)
- Conta Nível 2 (Circulante, Não Circulante, Patrimônio Líquido)
- Conta Nível 3 (Tipo da conta: Disponível, Realizável a Curto Prazo ou a Longo Prazo, Intangível, entre outros)
- Conta Nível 4 (Nome da conta: Fornecedores, Caixa e Equivalentes de Caixa, Simples Nacional, entre outros)

Nenhuma coluna foi adicionada.
